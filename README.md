# Physical Therapy Revenue Leakage & Patient Adherence Analysis

36% of patients fail to complete care, driving ~$78K in lost revenue. This project reflects a real-world healthcare analytics use case focused on improving outpatient rehabilitation outcomes and operational efficiency.

# 🔍 Problem

Outpatient physical therapy clinics often struggle with:
- high patient drop-off rates
- no-shows
- inefficient treatment planning

These issues lead to:
- lost revenue
- poor patient outcomes
- underutilized therapist capacity


# 📊 Dataset

This project uses a simulated outpatient physical therapy dataset designed to model:
- patient treatment plans
- visit adherence
- therapist scheduling
- no-show behavior


# ⚙️ Approach
- Cleaned and structured patient visit data
- Analyzed adherence trends across:
- injury types
- prescribed visit counts
- therapist performance
- Built a dashboard to visualize:
- drop-off rates
- no-show rates
- revenue leakage


# 📈 Key Insights
- 36% of patients fail to complete their plan of care
- 16% no-show rate contributes to operational inefficiency
- Estimated $78K in lost revenue from incomplete care plans
- Patient adherence declines as prescribed visits increase
- Drop-off rates are consistently high across all injury types
- Therapist-level variation suggests opportunities for targeted intervention


# 💡 Recommendations
- Implement reevaluation checkpoints to prevent unnecessarily long treatment plans
- Optimize scheduling strategies to reduce no-shows
- Develop therapist-specific performance monitoring
- Introduce patient engagement workflows to improve adherence
